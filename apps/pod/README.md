# Geo Hypergraph Template

This is a template for building Hypergraph apps. This template uses the following technologies:
- Hypergraph
- Geo Connect
- TanStack Start
- Tailwind
- Biome
- Bun
- TypeScript

## Getting started

Install dependencies using Bun

```sh
bun install
```

### Running the dev server

```sh
bun run dev
```

### Building the app

```sh
bun run build
```

## Using this template to spin up another project

```sh
bunx gitpick geobrowser/hypergraph-template/tree/main repo-name
cd repo-name
bun install
bun run dev
```
